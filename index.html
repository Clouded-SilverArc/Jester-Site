Documentation

Learn how to use Jester Bot

Getting Started

Invite the bot, set a prefix, and configure basic settings.

  • Use the Add to Server button on the home page
  • Default prefix is f!
  • Run f!help to see all commands
  • Use f!help <command> for detailed usage
  • Visit the Dashboard to configure permissions per role

Permissions

Control who can use which commands with the dashboard.

  • Go to Dashboard → select server → Permissions tab
  • Toggle commands on/off per role (9 categories, 107 commands)
  • Set channel-specific overrides to restrict commands
  • Use Allow All / Deny All for entire categories

Moderation (18 cmds)

Full suite of moderation tools.

  • f!ban, f!kick, f!mute, f!warn
  • f!purge — bulk delete messages
  • f!nuke — clone & delete a channel
  • f!hackban — ban by ID even if not in server
  • f!warnings / f!clearwarns to manage warns

Economy (24 cmds)

Full economy system with banking, gambling, and activities.

  • f!balance, f!daily, f!work, f!deposit, f!withdraw
  • Gambling: f!blackjack, f!slots, f!crash, f!roulette, f!coinflip
  • Activities: f!fish, f!hunt, f!scratch
  • Shop: f!shop, f!buy, f!inventory, f!additem

Fun (19 cmds)

Games, memes, and social commands.

  • Games: f!tictactoe, f!rps, f!trivia, f!numberguess
  • Social: f!ship, f!howgay, f!roast, f!compliment
  • Random: f!8ball, f!meme, f!joke, f!fact
  • Party: f!truth, f!dare, f!wyr

Giveaways (8 cmds)

Full giveaway system with advanced controls.

  • f!giveaway start <duration> <winners> <prize>
  • f!giveaway end / f!giveaway reroll
  • f!giveaway pause / f!giveaway resume
  • f!giveaway drop — instant giveaway, first to react wins
  • f!giveaway list — view active giveaways

Leveling (6 cmds)

XP system with rank cards and role rewards.

  • f!rank — view your rank card
  • f!leaderboard — server XP leaderboard
  • f!levelconfig — configure XP settings
  • f!setlevel / f!setxp / f!resetlevel (admin)

Utility (13 cmds)

Helpful utility commands for everyday use.

  • f!afk, f!remind, f!timer, f!poll
  • f!snipe / f!editsnipe — recover deleted/edited messages
  • f!calculate, f!color, f!embed
  • f!prefix — change the bot prefix

Text (11 cmds)

Text manipulation and conversion tools.

  • f!ascii — convert text to ASCII art
  • f!base64encode / f!base64decode
  • f!binary, f!emojify, f!mock
  • f!reverse, f!uppercase, f!lowercase
